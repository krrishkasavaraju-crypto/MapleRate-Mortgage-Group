# MapleRate-Mortgage-Group
MapleRate Mortgage Group is a modern, responsive mortgage brokerage demo website built to showcase clear service messaging, trust-focused design, and interactive user experience for a fictional Canadian mortgage brand. The project includes a dynamic mortgage calculator, testimonial and FAQ sections, mobile-friendly navigation, smooth scroll animations, floating stat cards, and a modal contact form to simulate a polished lead-generation website. It was built using HTML5, CSS3, and vanilla JavaScript, with custom responsive layouts, interactive UI behavior, and typography powered by Google Fonts using Playfair Display and DM Sans.
